In `extras`, you can share anything else with other modulesâ€”such as utility functions, configurations, and more.

```ts [module.config.ts]
export default {
  moduleName: 'my-module',
  extras: {
    getStatus() {
      return true;
    },
    menuEntries: [{
      label: 'Home',
      path: '/'
    }]
  }
}
```

To read this extra configuration, you can use the helper function `getExtras`.

```tsx [App.tsx]
import { getExtras } from 'reactmos'

export default function App() {
  const { menuEntries } = getExtras('my-module')
  return (
    ...
  )
}
```
